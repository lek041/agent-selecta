# 🎵 Agent Selecta v2.0

> Automatic audio file cataloger with a terminal UI — identifies artist & album via multiple APIs and organizes your music library automatically.

```
  ╔══════════════════════════════════════╗
  ║      A G E N T   S E L E C T A      ║
  ║          v 2 . 0   T U I             ║
  ╚══════════════════════════════════════╝
  AcoustID · MusicBrainz · Last.fm · Discogs · Deezer · Spotify DB
```

---

## What it does

Agent Selecta scans your source folders (SELECTA), identifies the artist and album of each audio file by querying multiple metadata APIs, rewrites the file tags, and moves everything to a structured library (ARCHIVE):

```
ARCHIVE / A / Artist Name / Album Name / track.mp3
```

---

## Requirements

- **Python 3.10+**
- **macOS 12+** or Linux (Ubuntu 20.04+)
- Internet connection (for external APIs)
- `fpcalc` — optional, for acoustic fingerprinting:
  ```bash
  brew install chromaprint        # macOS
  sudo apt install libchromaprint-tools  # Ubuntu
  ```

## Installation

```bash
# 1. Clone the repo
git clone https://github.com/lek041/agent-selecta.git
cd agent-selecta

# 2. Make the launcher executable
chmod +x run.sh

# 3. Run — venv and dependencies are created automatically
./run.sh
```

On first run, `run.sh` creates an isolated `.venv/` and installs all packages from `requirements.txt`. Subsequent runs skip the install step and open the UI directly.

## API Keys

Open `agent_selecta.py` and fill in the constants at the top:

```python
LASTFM_KEY    = "YOUR_KEY_HERE"    # https://www.last.fm/api/account/create
DISCOGS_TOKEN = "YOUR_TOKEN_HERE"  # https://www.discogs.com/settings/developers
ACOUSTID_KEY  = "YOUR_KEY_HERE"    # https://acoustid.org/login
```

> MusicBrainz and Deezer don't require an API key.
> APIs are optional — the system falls back automatically if one doesn't respond.

## Configure your folders

In `agent_selecta.py`, set your paths:

```python
PASTAS_ORIGEM = [
    "/Users/YOUR_USER/Music/SELECTA",   # source folder(s) — new files to catalog
]
PASTA_ARCHIVE = "/Users/YOUR_USER/Music/ARCHIVE"       # organized library
PASTA_UNKNOW  = "/Users/YOUR_USER/Music/Z_UNKNOW"      # unidentified files
PASTA_LOST    = "/Users/YOUR_USER/Music/Z_LOST"        # error logs
PASTA_REPORT  = "/Users/YOUR_USER/Music/Z_REPORTS"     # CSV reports
SQLITE_PATH   = "/Users/YOUR_USER/Music/spotify_agent.db"  # generated by converter_spotify.py
```

## Spotify database (optional but recommended)

The local Spotify database is the fastest and most accurate identification source. To generate it:

```bash
source .venv/bin/activate
python converter_spotify.py
```

This creates `spotify_agent.db` in the project folder. Re-run it whenever you want to sync new additions from your Spotify library.

## Modes

| Mode | Description |
|------|-------------|
| **▶ UPDATE** | Processes SELECTA automatically with no user interaction |
| **⌕ SCAN** | Confirms each file manually before moving |
| **↺ REVIEW** | Reorganizes ARCHIVE and reviews unidentified files |
| **⚙ AUDIT** | Checks and corrects ID3/Vorbis tags across the entire ARCHIVE |
| **✦ RESCUE** | Re-identifies files with similarity score < 0.4 |

## APIs & sources

| Source | What it provides | Priority |
|--------|-----------------|----------|
| Spotify DB (SQLite) | Artist & album from local Spotify library | 1st — fastest, offline |
| AcoustID + MusicBrainz | Identification by acoustic fingerprint | 2nd |
| Last.fm | Artist validation and album lookup | 3rd |
| Discogs | Additional artist verification | 4th |
| Deezer | Supplementary metadata | 5th |
| ID3/Vorbis tag | Artist name embedded in the audio file | Fallback |

## Troubleshooting

**`ModuleNotFoundError: No module named 'textual'`**
Always run via `./run.sh`. Never run directly with `python3` outside the venv.

**UI freezes after clicking a button**
The background worker is waiting for your input. Click **Confirm**, **Skip**, or **Edit** to continue.

**`spotify_agent.db` not found**
Run `python converter_spotify.py` with Spotify installed. Without this file the system uses only external APIs.
